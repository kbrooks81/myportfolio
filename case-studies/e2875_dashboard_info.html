Case Studye2875 Dashboard

Power BI • Data Modeling • DAX

Summary

e2875 is an operational analytics dashboard designed to give stakeholders fast, reliable insight into system activity and trends using a clean data model, purposeful measures, and decision-focused visuals.

Note on access: This dashboard was built using data and workflows that can not be shared publicly. This case study focuses on approach, modeling decisions, and redacted visuals.

At a glance

  • Platform: Power BI, ServiceNow, SharePoint, Power Automate, SQL
  • Core skills: Data modeling, DAX measures, Power Query transformations
  • Goal: Decision-ready reporting for operational + leadership stakeholders
  • Deliverable: A dashboard that answers common questions quickly and consistently

Problem

Stakeholders needed a clear, reliable view of operational performance without manually combing through exports or spreadsheets. Existing reporting required too much manual effort, and results were either not consistently presented or not presented at all.

Solution

A Power BI report centered on a clean model and a date-aware measure strategy. The report pages were intentionally structured to guide users from overview → drilldown, with slicers and visuals optimized for quick scanning and deeper investigation.

Key features

  • Modeled schema: sensible relationships between each report to ensure reliable data across scopes
  • DAX measures: KPIs and trends built for clarity and consistent reuse across visuals
  • Power Query shaping: refresh-friendly transformations to standardize source exports
  • Usable UX: slicers + drill patterns built around stakeholder questions and concerns

Screenshots

Challenges and tradeoffs

  • Performance vs flexibility: keeping measures reusable without bloating visuals
  • Data consistency: normalizing exports so definitions stay stable across refreshes
  • Data collection: compiling all necessary data sources into a single source of truth
  • Design clarity: balancing executive scan-ability with meaningful detail when needed
