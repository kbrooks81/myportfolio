React NativeMealToGo

Search + Maps Restaurant Discovery

Summary: MealToGo is a React Native app focused on fast restaurant discovery. It combines city-based search with a map-first experience so users can explore options quickly.

At a glance

  • Tech: Expo, TypeScript, Context API, Maps, Places
  • Role: Solo developer
  • Repo: GitHub

Problem
Finding a place to eat in a new area can be slow when apps bury important details. I wanted a lightweight flow: search a city, see the map view, then drill into a restaurant quickly.

Solution
A simple, responsive UI that supports: city search, map zoom to the selected city, and restaurant selection with quick context (name, rating, open/closed, etc.).

Key features

  • City search + location-based map zoom
  • Restaurant selection updates map + details cleanly
  • State invalidation when changing cities to avoid stale selections

Challenges + tradeoffs

  • Keeping map state and selected restaurant state in sync across tab navigation
  • Balancing responsiveness with API calls and caching strategies

Next improvements

  • Authentication + profile type settings
  • Favorites + offline-friendly caching
  • Performance optimizations for markers and lists
